## 📌 Sobre o projeto

Este projeto tem como objetivo explorar um dataset real do mercado de tecnologia brasileiro, utilizando técnicas de análise de dados para compreender relações entre variáveis como remuneração, experiência, formação e uso de tecnologias.

A análise foi conduzida com foco em exploração, visualização e interpretação dos dados.

## 🎯 Objetivos

- Analisar padrões de remuneração no mercado de TI  
- Investigar a associação entre formação e salários  
- Analisar diversidade e representatividade (gênero, raça)  
- Explorar fatores relacionados à progressão de carreira  

## 📊 Dataset

- Fonte: Pesquisa Datahackers 2022  
- Formato: arquivos Excel  
- Dados sobre profissionais de tecnologia no Brasil  

Variáveis analisadas incluem:

- Salário  
- Tempo de experiência  
- Escolaridade  
- Tecnologias utilizadas  
- Gênero e raça  
- Situação profissional  

## 🛠 Tecnologias utilizadas

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- SciPy (teste estatístico Qui-quadrado)

## 🔎 Etapas da análise

### 1. Exploração inicial dos dados
- Leitura dos arquivos  
- Verificação de tipos de dados  
- Identificação de valores nulos  

### 2. Tratamento dos dados
- Limpeza e padronização  
- Tratamento de valores ausentes  
- Ajustes de variáveis  

### 3. Análise exploratória
- Estatística descritiva  
- Comparações entre grupos  
- Identificação de padrões  

### 4. Visualização de dados
- Gráficos para interpretação dos resultados  
- Comparações entre variáveis  
- Distribuições e tendências  

### 5. Análise estatística
- Aplicação do teste Qui-quadrado  
- Avaliação de associações entre variáveis categóricas  

## 📈 Principais análises

- Distribuição de salários por experiência e perfil  
- Tecnologias mais utilizadas no mercado  
- Associação entre escolaridade e remuneração  
- Análise de diversidade (gênero, raça e PCD)  
- Comparação entre diferentes perfis profissionais  

## 🔍 Principais insights

- Profissionais com maior experiência apresentam maior concentração nas faixas salariais mais altas  
- Há diferenças observáveis na distribuição salarial entre grupos analisados  
- Determinadas tecnologias aparecem com maior frequência entre profissionais com maiores salários  
- A formação contínua está associada a melhores faixas de remuneração  
- A representatividade de grupos minorizados ainda é reduzida no setor  

## 💡 Conclusão

A análise evidencia padrões relevantes no mercado de tecnologia brasileiro, especialmente em relação à remuneração, formação e uso de tecnologias.

Também foram observados desafios relacionados à diversidade e inclusão, indicando oportunidades de evolução no setor.

Por se tratar de uma análise exploratória, os resultados indicam associações nos dados, sem estabelecer relações causais.

## ⚠️ Limitações

- A análise é descritiva e exploratória  
- Não foram aplicados modelos preditivos  
- O dataset pode não representar igualmente todos os perfis profissionais  
- Não foi analisada causalidade entre as variáveis
